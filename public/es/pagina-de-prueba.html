Página de Prueba para Programas de Posgrado en Lingüística

Propósito

Esta página de prueba valida la funcionalidad del script generate_page_json.js.

Metadatos Requeridos

Esta página incluye todos los metadatos requeridos:

  • Bloque de comentario SEO_INTENT con keyword, audience, last_reviewed
  • Título (50-60 caracteres)
  • Meta descripción (140-160 caracteres)
  • Metadatos path_en y path_es
  • Enlaces hreflang (bidireccionales, auto-referenciales, x-default)
  • Jerarquía de encabezados apropiada (H1 → H2)
  • Elementos HTML semánticos
